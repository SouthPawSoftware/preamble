Preamble is a powerful JavaScript testing framework that runs in any modern HTML5 compliant browser. Preamble can be run headless via PhantomJS. Preamble has a small footprint. Preamble has no additional dependencies on other libraries or frameworks. Preamble has a fast and powerful testing engine. Preamble supports both TDD and BDD style semantics. Preamble has thorough reporting capabilities through the browser. Preamble makes the task of authoring and running tests easy, intuitive and even fun.
